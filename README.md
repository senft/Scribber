#Scribber

**Scribber** is a simple text editor that focuses on minimalism. (Screenshot at the bottom)


# Features:

* Highlight the current sentence
* Markdown syntax hilighting
* Minimalistic UI that lets you focus on writing your text
* Export to PDF

# Dependencies:

* GTK+
* pdflatex
* python-markdown
